Exception--The specified child already has a parent.

1.此异常出现在对ViewPager设置Adapter的时候，在第二次设置的时候在instantiateItem函数中container调用addview，由于第二次调用意在刷新，并且该view不是new 出来的对象，由于用的是之前的view，所以在addView(view)的时候由于该view已经有parent，再次进行addview会抛出此异常。此时需要在调用addview()之前进行如下判断：
ViewGroup parent = (ViewGroup) view.getParent();
if (parent != null) {
parent.removeAllViewsInLayout();
}
上段代码先判断是否有parent有的话移出所有view，然后再进行addView(view)既能解决该异常。
